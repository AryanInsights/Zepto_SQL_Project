# 🛒 Zepto Sales Analysis using SQL

## 📌 Project Overview

This project analyzes a realistic Zepto grocery sales dataset using SQL to extract meaningful business insights. It demonstrates core SQL concepts ranging from basic filtering to advanced analytical queries, making it suitable for beginner to intermediate Data Analyst portfolios.

The project is designed using PostgreSQL syntax and is largely compatible with MySQL with minor syntax adjustments.

---

## 🎯 Project Objectives

* Analyze sales performance across different product categories.
* Identify top-performing products and brands.
* Compare revenue across different cities.
* Analyze discounts and customer purchasing behavior.
* Practice SQL concepts commonly used in Data Analyst interviews.

## 📊 Key Insights

* Revenue varies significantly across cities.
* Some product categories consistently generate higher sales.
* High discounts do not always result in higher revenue.
* Product rankings change depending on the analytical function used.
